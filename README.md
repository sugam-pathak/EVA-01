# 🧪 EVA System HUD - Chrome Extension

A Neon Genesis Evangelion inspired HUD overlay for your browser. Makes your web browsing feel like you're piloting an EVA unit.

## ✨ Features

- **Cinematic Boot Sequence** - Watch the system initialize with typing effects and glitches
- **Real-time Clock** - Display current time in the HUD
- **Live System Stats** - Sync rate updates randomly like a real EVA unit
- **Spark Effects** - Particles burst out when you hover over any HUD element
- **Toggle Button** - Turn HUD on/off whenever you want
- **MAGI Status** - Shows all three MAGI supercomputers status
- **Alert Feed** - Emergency warnings and system notifications
- **A.T Field Monitor** - Watch your absolute terror field status
- **Sound Effects** - Warning sounds for emergency situations (optional)

## 🎮 Commands

Type these in the AI command input:

| Command | What it does |
|---------|---------------|
| `time` | Shows current time |
| `status` | Displays system status |
| `clear` | Clears the output |
